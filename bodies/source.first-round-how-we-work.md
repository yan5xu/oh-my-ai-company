# First Round How We Work

URL: https://www.firstround.com/how-we-work

采集时间：2026-07-21。官方投后合作说明。页面把支持集中在招聘、公司定位、建立可重复收入和下一轮融资，并称团队通常每 4–6 周围绕一个关键问题开展 working session；First Round 常在最初阶段占一个 board seat，也可能在后续投资者加入后让出。页面还称 LP 资本多数来自大学捐赠、医院和慈善机构，不接受主权财富或政府资本。

这是机构服务设计的 S1 证据，不是每家被投公司都实际获得同等频率、同等效果服务的证明。
