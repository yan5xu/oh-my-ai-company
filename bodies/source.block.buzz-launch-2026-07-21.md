# Block official Buzz launch

Block 于 2026-07-21 发布 Buzz，称其为免费、开源、model/agent-agnostic 的 human-agent collaboration platform，覆盖 channels、threads、DMs、voice、media、code repositories 与 workflows，可连接 Claude Code、Codex、goose 或自带 agent。

官方核心差异是 agent 拥有独立 cryptographic identity 和明确权限，而不是冒用人的账号；团队可自托管 relay，也可使用 Block-hosted 版本。Bradley Axen 以 Head of AI Capabilities 身份给出产品立场。

这是 Block 的一手 launch 口径，可支持产品定位、发布主体和公开功能范围；不能独立支持市场采用、ROI、客户数或产品成熟度。
