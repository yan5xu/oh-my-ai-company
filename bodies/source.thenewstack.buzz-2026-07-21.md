# The New Stack coverage

The New Stack 采访 Bradley Axen，补充了几项官方细节：

- agent 使用独立 keypair，并通过额外签名绑定 human owner；Block 将该身份方案提交给 Nostr，但尚未获 upstream 接受。
- hosted relay 在 launch 阶段免费、Axen 称为 beta；当时尚未设统一 usage limits，enterprise-grade Slack replacement 未来可能需要不同安排。
- Block 仍在内部使用 Slack，Buzz 与 Slack 可并行；Axen 说能否完整替代 Slack 需要时间。
- 报道将 Buzz 放在 Block 更广的 agent stack 中，但 BuilderBot 的 200k operations/day、1,500 PRs/week、约 15% production changes 是 BuilderBot 证据，不能归到 Buzz。

文章发布时称 Buzz repo “just over 100 stars”，同日随后 GitHub 页面约 2.1k，说明 launch 传播导致数值快速变化。
