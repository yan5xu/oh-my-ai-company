# First Round Angel Track

URL: https://www.firstround.com/angel-track/

采集时间：2026-07-21。Angel Track 是 First Round 的天使投资人培养与社群项目，官方称已有 400 多位 alumni，课程覆盖 sourcing、评估、服务创始人和 portfolio construction。页面还明确提到 alumni 会填补相关融资轮次并向 First Round 推荐创始人。

该页直接支持“社群可以形成案源与联合投资网络”的机制判断，但没有公开 referral 数量、命中率、投资回报或利益冲突安排。
