文章解读 MCP Enterprise-Managed Authorization（EMA）在 2026-06-18 进入 stable：企业可由 IdP 集中配置 MCP server 访问，通过 SSO 取得 identity assertion，再换取各 server token，避免每位员工逐个完成 OAuth consent。作者称 Anthropic、Microsoft、Asana、Atlassian、Figma、Okta 等已实现，NewCore 也支持。

文章进一步提出 Agent 数量与身份事件会达到人类的约 100 倍、人类活动低于 1%。这些是 NewCore 对未来的预测，不是独立行业统计。[[company.newcore]]
