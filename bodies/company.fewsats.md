# Fewsats

Fewsats 是 Jordi Montes 创办的 machine-native payment infrastructure，围绕 L402/x402 让软件与 AI agents 通过 HTTP 按调用访问付费资源。

2026-06-11，[[company.sapiom]] 宣布收购 Fewsats，Jordi 加入 Sapiom 任 founding engineer。交易金额未披露。Fewsats 的公开 GitHub 资产包括 `awesome-L402`、`amazon-mcp`、`proxy402`、`fewsats-mcp` 和 Python SDK，说明此次交易包含协议与开发者工具积累，而不只是人物履历。

证据：[[source.sapiom.fewsats-acquisition-2026-06-11]]、[[person.jordi-montes]]。
