# Exa MCP Server

S1 official source, fetched 2026-07-08 via `pinixc browser read`.

Facts:
- Exa MCP connects Claude, Cursor, VS Code and other AI tools to Exa search, code search, and research capabilities.
- The page says it is free and open-source.
- It provides a remote MCP URL: `https://mcp.exa.ai/mcp`.
- Example config for Cursor uses:
  `{ "mcpServers": { "exa": { "url": "https://mcp.exa.ai/mcp" } } }`

Interpretation: MCP is a distribution mechanism: Exa can become the default web/search tool inside coding agents and desktop AI assistants without every developer writing API integration.
