# Agent 平台上的成品助理入口

横向 Agent 平台不再只给用户一张空白 Builder，而是把 inbox、日历、会议、消息等高频任务组合成一个默认可用的成品助理。用户先获得具体价值，平台再通过长期上下文、连接器和自定义 workflow 承接更复杂的需求。

[[company.lindy]] 是当前清晰样本：2023–2025 年强调 AI employee / builder，2026 年把 Lindy Assistant 设为前台入口，底层仍保留 Custom Agents、Computer Use、Evals 与企业治理。[[source.lindy.assistant-launch-2026]] [[source.lindy.docs-agent-builder]]

待验证的是两种用户心智是否能共存：个人用户可能不愿学习 workflow，企业买家也可能认为个人助理定位过轻。
