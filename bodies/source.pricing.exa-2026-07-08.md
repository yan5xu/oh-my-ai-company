# Exa pricing page snapshot

S1 official pricing page, fetched 2026-07-08 via `pinixc browser read`.

Facts:
- Endpoint pricing listed: Search $7 / 1k requests, Deep Search $12 / 1k, Deep-Reasoning Search $15 / 1k, Contents $1 / 1k pages, Monitors $15, Answer $5.
- Additional result above 10 costs $1 / 1k requests for most endpoints.
- AI page summaries $1 / 1k pages.
- Agent pricing: default effort auto; fixed modes include Minimal $0.012/request, Low $0.025, Medium $0.10, High $0.50, X-high $1.00.
- Usage components: Agent Compute Units $0.10/ACU, search tool calls $0.005/search, email enrichment $0.02/email, phone enrichment $0.07/phone.
- Startup and education grants: $1,000 worth of free credits.

Interpretation: Exa’s pricing exposes an important category tension: search for agents can be low-latency API pricing or high-compute agent/deep research pricing.
