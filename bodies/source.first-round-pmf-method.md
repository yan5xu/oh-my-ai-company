# First Round PMF Method

URL: https://www.firstround.com/pmf

采集时间：2026-07-21。PMF Method 是 First Round 面向早期 B2B 创始人的免费项目，官方称不要求股权、也不提供资本，覆盖 customer discovery、design partners、positioning、product iteration 和 founder-led sales。页面同时写明，项目让创始人深入了解 First Round，也让机构更了解创始人。

它因此既是教育/支持项目，也是双方在投资前建立关系和观察的入口。后一句属于我们的机制推断，公开资料没有披露项目转化为投资的比例。
