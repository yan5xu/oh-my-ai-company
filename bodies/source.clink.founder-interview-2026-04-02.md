# Harness Payment，开启 Agent 的自主支付时代：Clink 创始人专访

腾讯新闻“十字路口”2026-04-02 发布，对 [[person.patrick-wu-clink]] 的直接访谈。

## 访谈披露

- 团队十几人，另使用几十个 Agent。
- 已融资数百万美元；上一轮由 Celtic 与 BV 百度风投联合领投。
- 团队 2025 年初开始做支付，先补 PCI、订阅、路由和税务。
- 第一位商业客户在 2025 年 9-10 月，是一家游戏公司。
- Agent Payment 在 2026 年春节后加速发布。
- 核心不是给 Agent 无约束支付能力，而是预算、频率、场景、熔断与 human-in-the-loop。
- 产品选择法币、卡与本地支付，而不是稳定币优先，因为希望沿用普通用户熟悉的体系。
- 早期 Agent 客户案例包括 PollyReach、ModelMax/Moxt 相关场景。
- 团队声称人均每日消耗 5,000 万 token，并用多个 coding agent 分工 planning、coding、review、test；此为创始人口径，未独立验证。

## 证据边界

这是直接访谈，适合确认创始人观点、公司历史和融资口径，但融资金额、客户与内部效率仍需要投资方、客户或运营数据交叉验证。
