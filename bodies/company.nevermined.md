# Nevermined

Nevermined 面向 AI agents 提供 meter、access、pricing 与 settlement，公开资料支持 usage plans、fiat/crypto、x402、MCP 和 A2A。

在 [[company.sapiom]] 的竞品图里，它属于直接竞争：双方都控制能力调用的计量、访问和结算；Nevermined 更偏 Agent monetization，Sapiom 当前更偏 capability access + execution engine。

本对象目前是竞品 seed，尚未完成独立深调。证据：[[source.nevermined.official-product-2026-07-14]]、[[note.sapiom-competitor-map-2026-07-14]]。
