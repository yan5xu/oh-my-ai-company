# Multica 支持的 AI 编程工具

**证据等级：S1，官方文档。**

截至 2026-07-15，文档列出 15 种 provider/runtime：Antigravity、Claude Code、CodeBuddy、Codex、Copilot、Cursor、Hermes、Kimi、Kiro CLI、OpenCode、DevEco Code、OpenClaw、Pi、Qoder、Trae CLI。

Agent 可配置 provider、runtime、instructions、环境变量、参数、MCP、并发数和 Skills。该证据说明 Multica 是 runtime-agnostic 管理层，而非自研模型。官网仍写 14，属于口径冲突。关联：[[company.multica]]。
