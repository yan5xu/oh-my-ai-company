# Endor Labs homepage / AURI

S1 official website, fetched 2026-07-08 via `pinixc browser read`.

Facts:
- Endor Labs positions itself as an agentic application security platform for AI-generated and human-written code.
- AURI combines agentic reasoning with deterministic program analysis to verify what is real, ignore false positives, and deliver fixes.
- Homepage metrics: 10x fewer security tickets, 83% fewer blocked PRs, 6x faster fixes.
- AURI is described as security intelligence for agentic software development.
- Core design: AI coding agents should not verify security of their own code; AURI is an integrated but independent policy/enforcement layer across AI coding agents.
- Integrations: Hooks, Skills, MCP, CLI.
- Verification: data flow, call paths, reachability, exploitability analysis, contextual fixes.
- Reproducibility/auditability: traceable, repeatable decisions, audit-ready evidence.
- Research team backgrounds mentioned: Amazon, Cisco, Meta, GitHub, Microsoft, NGINX, Palo Alto Networks, Splunk, Uber; publications in ICSE, ASE, IEEE TSE, Empirical Software Engineering.

Interpretation: Endor/AURI is a security verification substrate for AI coding agents, adjacent to coding agent workflow but deliberately decoupled from code generation.
