# Temporal GitHub org

S1 official GitHub org, fetched 2026-07-08 via `pinixc browser read`.

Visible facts:
- Open-source durable execution system.
- Temporal abstracts away scalable/reliable distributed system complexity by preserving complete application state and recovering across crashes/service outages.
- Eliminates custom code for timers, event sourcing, state checkpointing, retries, and timeouts; can reduce need for queues/pubsub/schedulers.
- Temporal adopters listed: Stripe, Netflix, Datadog, HashiCorp, Alaska Airlines, Box, and more.
- Claimed usage: every message on Twilio, every Coinbase transaction, and every Snap story uses Temporal.

Interpretation: strong open-source adoption evidence, though usage claims are from the GitHub org copy and should be treated as official marketing claims.
