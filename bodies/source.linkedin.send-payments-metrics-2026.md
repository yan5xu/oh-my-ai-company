# Send Payments CEO：内部 AI Agent 组织与运营指标

Ryk Neethling 公开称 Send Payments 运行的 AI Agents 数量与人类员工相当，每个 Agent 有 reporting line、job description 和 measured hours。

归因边界：该帖没有把所有指标单独归因给 Relevance AI；它与另一条平台采用帖共同证明客户的 Agent 运营实践，但不能直接写成供应商 ROI。
