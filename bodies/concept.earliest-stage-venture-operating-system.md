# 最早期风投操作系统

这不是“投后服务很多”的同义词，而是机构围绕 pre-PMF 阶段，把以下环节做成相互连接、可重复的系统：

1. 投资判断：共同评估、partner 专长分工、rubric、偏差控制和事后复盘；
2. 组织补位：早期招聘、定位、GTM、客户和融资；
3. 工作机制：围绕具体难题的 working session，而非只看 board reporting；
4. 上游网络：在正式融资前通过 founder program、angel program 和内容建立关系；
5. 知识回流：把 portfolio 与 operator 的经验重新变成内容、课程和下一轮判断资产。

[[investor.first-round-capital]] 是当前首个完整样本。PMF Method、Angel Track、The Review、Applied Intelligence 与 Pitch Assist 共同构成其系统，但每个模块是否提高融资、招聘、收入或基金回报，公开证据仍不足，因此本概念 confidence 为 medium。

不要把所有“有 platform team 的 VC”都归入该概念。最低判断门槛应是：阶段专注明确，投资决策与投后服务可见，且内容/社群与案源或公司支持之间存在事实连接。
