# Kylon Developer Docs index

官方 `llms.txt` 将 Kylon 的 developer surface 分成 CLI gateway、workspace CLI 与 proxy APIs。资源边界包括 workspace、agent、channel、gateway、provider、API key；workspace CLI 覆盖 channels、messages、threads、tables、files、workflows、web projects 和 secrets；Proxy 覆盖工具搜索/授权/执行、模型与 provider API、billing、authentication 和 error contract。该索引证明产品已有系统化开发接口，不证明这些接口已有外部客户规模。
