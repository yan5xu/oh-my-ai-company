# Mastercard launches Agent Pay for Machines

Mastercard 发布 Agent Pay for Machines（AP4M），组合 credentialing、permissioning、transacting 与 cards/accounts/stablecoins multi-rail settlement。Sapiom 是 30 多个首批参与者/支持者之一，名单还包括 Skyfire、Nevermined、Stripe、Coinbase、Cloudflare 等。

Ilan Zerbib 的官方引语把分工描述为 Mastercard 提供 settlement，Sapiom 提供 runtime、execution 和 access。这支持 Sapiom 从支付网关上移到执行层的定位。

证据边界：S1 Mastercard 公告。它证明生态参与与产品方向，不证明独家合作、客户转化或投资关系。
