# Monid Terms of Use snapshot

2026-07-22 官方 app 引用的 Termly Terms 包含：

- Services / Content 按 “AS IS” 提供，并保留中断、修改与终止权。
- Refunds section 要求查看另行发布的 Return Policy，但本轮未从产品找到明确 return-policy 页面。
- Prohibited Activities 同时禁止 automated use、data mining/robots、automated account creation，以及使用 buying agent / purchasing agent 购买。
- 争议使用 AAA binding arbitration；适用 California law。

关键冲突：Monid 的产品定位就是让 Agent 自动 discover、run 和 pay，但通用 Terms 模板仍禁止 automated use 与 buying agent。除非产品另有书面授权或新版条款，这构成 contract ambiguity，不能假定 marketing copy 自动覆盖法律条款。

证据边界：S1 法律文本快照，不构成法律意见。
