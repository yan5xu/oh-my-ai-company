# YC: Kernel

URL: https://www.ycombinator.com/companies/kernel

Evidence: S1 official YC profile.

- YC S25；2025 年成立。
- 创始人：[[person.catherine-jue]] 与 [[person.rafael-garcia]]。
- One-liner: “Crazy fast, open source infra for AI agents to use the Internet.”
- 早期页面强调约 300ms launch/connect、isolated VMs、live views/replays、agent runtime、session reuse、residential proxies、CAPTCHA 与把 agent 托管成 HTTP endpoint。
- YC 页面显示 6 employees，但 [[source.blog.kernel-vision-2026-07-10]] 在 2026-07-10 明确写团队已到 20 人，应以后者为当前口径。
- Raf 曾创办 Clever（YC S12，2021 年约 $500M exit）；Catherine 曾创办 Sway Finance（YC S16），后在 Cash App 领导 FDE 与 docs platform。
