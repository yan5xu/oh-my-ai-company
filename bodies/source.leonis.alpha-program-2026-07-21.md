# Leonis Alpha Program

[[investor.leonis-capital]] 官方 Alpha Program 页面面向 technical core 已可验证、但公司与市场仍很早期的创始人。页面公开：

- 10 万美元，founder-friendly terms；
- 决策可在数日内完成；
- 团队以“founding COO”方式参与 company setup、产品方向、technical pilots、网络和 seed preparation；
- 强调 hands-on，而不是只提供资本。

页面 FAQ 在本轮浏览时没有稳定渲染完整答案，因此未补条款、股权比例、cohort、录取率或后续融资数据。该 source 证明 program 设计与公开承诺，不证明支持效果。
