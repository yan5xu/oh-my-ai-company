# Felicity customer story

URL: https://www.kernel.sh/customers/felicity

Evidence: S1 vendor-published customer case.

Felicity 是 healthcare AI-native RPA 平台，使用 browser agents 处理 EHR、referrals、patient charts 等无稳定 API 的流程。Kernel 官网称迁移后 automation 速度提升 2 倍，并减少每月数小时 incident debugging。该案例也解释了 Kernel 投入 ISO 27001、HIPAA 与 BAA 的客户动因。
