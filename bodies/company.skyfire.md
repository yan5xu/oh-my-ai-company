# Skyfire

Skyfire 是 Agent commerce/payment control plane。官方产品面覆盖 KYA identity、agent account/wallet、pay token/JWT、buyer/seller services 与 settlement。

在 [[company.sapiom]] 的竞品图里，它属于直接竞争：双方都争夺 Agent 身份、授权和支付控制面；Skyfire 更集中于 commerce/wallet/token，Sapiom 当前把这一层与 capability marketplace 和 execution runtime 捆绑。

本对象目前是竞品 seed，尚未完成独立深调。证据：[[source.skyfire.official-product-2026-07-14]]、[[note.sapiom-competitor-map-2026-07-14]]。
