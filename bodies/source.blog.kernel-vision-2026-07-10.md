# Building for the Internet's trillionth user

URL: https://www.kernel.sh/blog/trillionth-user-vision

Author: [[person.catherine-jue]] · Published: 2026-07-10 · Evidence: S1 official vision.

- 团队在 16 个月内从 2 位创始人增长到 20 人。
- 官方称 2026 年已启动 tens of millions of browsers。
- 三个 north stars：为 agent 提供完美 web infrastructure；推进速度/可靠性/成本前沿；赋能 legitimate agents，并让每个 agent 的能力复利。
- legitimate agent 被定义为获得 end-user consent/delegation，授权 explicit、traceable、revocable，且网站可识别和审计。
- 文章认为 agent identity 应可跨设备和组织 portable。
- 这是战略方向与公司口径，不是标准采用结果。
