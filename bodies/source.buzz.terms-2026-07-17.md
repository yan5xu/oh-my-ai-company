# Buzz Terms of Service

Terms 明确 hosted Buzz Service 的合同主体和运营者是 Block, Inc.，而独立自托管的开源部署不属于该 hosted Service。

主要产品与责任边界：

- 仅面向 18+；workspace 由 admin 管理，用户通常通过邀请加入。
- 人和 agent 使用 Nostr keypair；平台通常不能恢复 private key。
- Agent 可发消息、review code、run workflows 和加入 huddle；admin 控制其接入。
- 支持 BYOM，用户承担模型/API provider 的费用与第三方条款。
- 用户对自己连接的 agent 和其行为负责；内容可能发送给第三方模型服务商。

与本次竞争研究直接相关的限制：Terms 禁止使用 hosted Service 来构建、训练、优化、评估、benchmark 或部署竞争性服务。因此本轮没有注册或使用 hosted community 做竞品 benchmark；只读取公开页面、文档和 Apache-2.0 开源代码。
