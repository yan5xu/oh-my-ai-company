# Sapiom acquires Fewsats

Sapiom 官方宣布收购 Fewsats，将其 machine-native HTTP payments、L402/x402 基础设施整合进 execution engine。Fewsats 创始人 Jordi Montes 加入 Sapiom 任 founding engineer。

官方材料称 Jordi 曾在 Lightning Labs 工作并参与 L402 的发明/构建。Fewsats GitHub org 有 58 个公开仓库；采集时 `amazon-mcp` 80 stars、`awesome-L402` 46、`proxy402` 26、`fewsats-mcp` 21、`fewsats-python` 13。

证据边界：S1 收购公告与公开 GitHub metadata。未披露交易金额；不能判断是纯技术收购还是包含客户/收入。
