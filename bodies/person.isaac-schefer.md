# Isaac Schefer

Isaac Schefer 是 [[investor.leonis-capital]] 当前官网列出的 CFO。公开 profile 显示他与 Nakama Venture Solutions 相关，并有 Carta senior project manager 经历。

他的职责支持 Leonis 已有专门的基金财务与运营能力，而不是由投资合伙人完全兼任后台管理。[[source.leonis.team-2026-07-21]]

官网未将 Isaac 列为 Partner，也没有公开证据显示他参与投资判断、IC 投票或 portfolio support，因此结构化角色记为 operator，只表达财务运营关系。
