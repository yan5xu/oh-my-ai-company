# Sapiom governance 与 rules 文档

公开 rules 可限制：每次 run、每日、每周、每月 spend；API call 数；token 数；per-service usage；rate。平台记录 transaction 与 rule evaluation log。

这些控制可回应 Agent 失控重试和预算超支的一部分风险，但本轮公开文档未清楚看到：新 vendor/capability 单独审批、vendor category policy、不可逆交易恢复、完整 KYA/risk policy 和 dispute handling。

证据边界：S1 官方文档，只能说明公开控制面，不说明所有企业能力或 roadmap。
