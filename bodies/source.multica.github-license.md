**证据等级：S1，LICENSE 原文。**

许可并非标准 Apache 2.0：

- 允许内部商业使用；
- 作为第三方托管、嵌入式商业服务、SaaS、managed service 或 integrated commercial offering 提供时，需要商业许可；
- 前端 logo/copyright 不可移除；
- 协议方保留未来调整协议的权利；
- 贡献代码可被项目方商业化。

GitHub API 的 SPDX 为 `NOASSERTION`。因此更准确的分类是 source-available、可自托管、带商业限制，而不是标准 OSI open source。关联：[[company.multica]]。
