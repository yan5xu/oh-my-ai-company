# 硅谷风投家将注资中国科技创业公司
# 纽约时报中文网：Accel、Jim Breyer 与 IDG 中国关系

报道记录 2014 年 Jim Breyer 与 IDG 共同参与一只 $586M 中国基金，并追溯双方自 2005 年开始的合作。它说明中文世界长期把 Accel 的中国影响力理解为 **Breyer 与 IDG 的关系网络**，而非 Accel 在中国独立复制美国团队。

名称也值得注意：文章把 Accel Partners 译作“阿克塞尔合伙公司”。当前中文媒体通常直接写 `Accel` 或 `Accel Partners`；不应把 Accel 译成“光速”，后者通常指 Lightspeed Venture Partners。

报道为 2014 年历史快照，不代表 Accel 当前在中国的组织与投资状态。
