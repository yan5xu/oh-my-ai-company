# Ilan Zerbib

[[company.sapiom]] 创始人兼 CEO。此前创办 consumer price-protection 产品 Earny，并在 2021 年退出；之后在 Shopify Payments 任工程总监近五年。

Sapiom 官方材料称，他参与把 Shop Pay 扩展到超过 `$100B GMV`，并构建 Shop Cash。该履历与 Sapiom 从 Agent 支付、授权和成本控制切入高度一致。

2026-07-02，他提出 Agent infra 的 “horseless carriage test”：不要只把人的 SaaS workflow 套上 Agent，而要寻找机器原生执行路径。该观点帮助解释 Sapiom 从 gateway 向 runtime 上移。
